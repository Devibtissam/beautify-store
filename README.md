Beautify is a complete online shop. Built from scratch using ReactJS & Redux and pure CSS for performance, Strapi Api for the Backend, it has Search by Category functionality.

## Tech Stack

```
core : 
- React js
- Redux js
- SCSS
- Strapi CMS

communication tool: 
- axios

```
